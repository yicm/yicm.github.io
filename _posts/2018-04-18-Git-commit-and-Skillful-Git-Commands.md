## 1 两步走实现代码提交

使用`git add`和`git commit`两个命令就可以实现本地代码的提交。

```
# $ git add -A
$ git add FILENAME
$ git commit -m "your comment for this commit."
```

## 2 初步熟悉Git命令

```
$ git clone 
$ git init
$ git status
$ git log
$ git checkout 
$ git remote 
$ git branch 
$ git diff
$ git reset
$ git mv
$ git rm
```
